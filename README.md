Репозиторий курса по эргатическим системам

## Структура репозитория

```
IU1-41M-2026/
├── lab1/        # Логистическая регрессия, переобучение
├── lab2/        # AutoML, классификация ЭКГ
├── lab3/        # Спектральный анализ ЭЭГ, вейвлет-преобразование
├── lab4/        # Свёрточные сети для классификации ЭЭГ (эпилепсия)
├── lab5/        # Нейросети для распознавания когнитивных состояний
└── project/     # Проектная сессия
```

Внутри каждой папки — папки студентов вида `firstname_lastname/`.

---

## Как сдать работу

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/ergaticsystems/IU1-41M-2026
cd IU1-41M-2026
```

### 2. Создайте свою ветку

Имя ветки — `labN/firstname_lastname`, например:

```bash
git checkout -b lab1/ivan_petrov
```

### 3. Создайте папку и добавьте файлы

Создайте папку с вашим именем внутри нужного задания и положите туда файлы:

```
labN/
└── firstname_lastname/
    ├── report.pdf
    └── notebook.ipynb   # (не требуется только для project/)
```

### 4. Сделайте коммит

```bash
git add lab1/ivan_petrov/
git commit -m "lab1: ivan_petrov"
```

### 5. Отправьте ветку и создайте Pull Request

```bash
git push origin lab1/ivan_petrov
```

Откройте Pull Request в GitHub: `lab1/ivan_petrov → main`.  
В заголовке PR укажите: `Lab1 — Ivan Petrov`.

### 6. Уведомите преподавателя

После создания Pull Request напишите в Telegram преподавателю и укажите:
- номер лабораторной работы,
- имя и фамилию на латинице,
- либо просто скиньте ссылку на вашу папку в репозитории.

Работа считается сданной только после отправки уведомления.

## Требования к файлам

| Файл | Требования |
|---|---|
| `report.pdf` | Отчёт в формате PDF, описывает методологию и результаты |
| `notebook.ipynb` | Jupyter Notebook с рабочим кодом, все ячейки выполнены |

- Папка называется строго `firstname_lastname` на латинице в нижнем регистре (например: `ivan_petrov`).
- Не кладите в репозиторий датасеты, модели и другие большие файлы — только отчёт и ноутбук.
- Проверьте, что ноутбук запускается без ошибок перед сдачей.
